netstat -ano 	--> To get the active running port on the system

netstat -ano | findstr : <port number> 	--> To find the specific port from the system.
					--> This command will return PID.

taskkill /PID <PID Number that return> /F --> It will deactivate the port.



# To change the port in Spring

- Under src/main/resources open the file names application.properties 
- Add this line "server.port=<port Number>
